📖 关于本站

NFLSHC Chat 是一款基于 GitHub Issues 构建的轻量级聊天平台。我们致力于为用户提供安全、私密、功能丰富的聊天体验，无需复杂的服务器部署，即可享受群聊、私聊、文件分享等多种功能。

本平台采用现代化的 Web 技术栈，支持 Markdown 格式消息、实时预览、消息收藏、聊天记录导出等功能，让您的沟通更加高效便捷。

✨ 主要功能

💬

群聊 & 私聊

支持创建群聊和一对一私聊，满足不同场景需求

📝

Markdown 支持

消息支持 Markdown 格式，实时预览效果

⭐

消息收藏

收藏重要消息，方便日后查阅

🔍

消息搜索

按关键词、发送者、时间范围搜索历史消息

📤

数据导出

导出聊天记录为 JSON 或 HTML 格式

📊

群聊统计

消息趋势、发言排行榜、词云分析

@

@提及通知

使用 @用户名 或 @所有人 通知其他成员

🔒

隐私保护

数据存储在 GitHub Issues，完全透明可控

🛠️ 技术栈

HTML5 CSS3 JavaScript (ES6+) GitHub API Marked.js (Markdown 渲染) Chart.js (数据可视化) GitHub Pages GitHub Actions
